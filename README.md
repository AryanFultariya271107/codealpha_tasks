# CodeAlpha Internship Projects

Welcome to my CodeAlpha Internship Projects repository. This repository contains the machine learning and data analysis projects completed during my internship at CodeAlpha. These projects demonstrate my skills in Python, data preprocessing, exploratory data analysis, visualization, and machine learning model development.

---

## Projects Included

### 1. Car Price Prediction

#### Overview

A machine learning project that predicts the selling price of a car based on various features such as year, fuel type, transmission, ownership history, and other vehicle specifications.

#### Features

* Data preprocessing and cleaning
* Feature engineering
* Exploratory Data Analysis (EDA)
* Machine Learning model training
* Price prediction for new car data
* Model performance evaluation

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

### 2. Iris Flower Classification

#### Overview

A classification project that predicts the species of an Iris flower using measurements such as sepal length, sepal width, petal length, and petal width.

#### Features

* Data visualization
* Feature analysis
* Classification model development
* Species prediction
* Model evaluation using accuracy metrics

#### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
